# Intellivision-composite
Intellivision I/II composite mod

2-stage composite signal amplifier with transistor.

**Components:**

R1: 51K

R2: 33K

R3: 600R

R4: 330R

R5: 15K

R6: 22K

R7: 75R

R8: 22R

C1,C2,C3,C4: 100uF 16v

Q2,Q2: 2N3904
